Politique de confidentialité

Vos données, ici.

DERNIÈRE MISE À JOUR · [À DATER]

Ce document explique, sans jargon, ce que je fais des informations que vous me confiez — que ce soit via le site ou en séance.

01Responsable du traitement

Le responsable du traitement des données est Geoffroy Bertrand, éditeur du site Flex Alignement. Pour toute question relative à vos données : geoffbertrand@gmail.com.

02Données collectées via le site

Via les formulaires de contact, les données suivantes peuvent être collectées :

  • Formulaire entreprise : nom, fonction, organisation, taille de l'organisation, e-mail, téléphone (facultatif), nature de la demande, description libre, délai.
  • Formulaire particulier : prénom (facultatif), e-mail, type d'accompagnement, description libre, format préféré.

Le site n'enregistre pas de données de navigation identifiantes. Aucun cookie publicitaire ni tiers de suivi comportemental n'est utilisé. Une mesure d'audience anonyme, sans cookie peut être utilisée ([Plausible / Umami selon ce qui sera en place]).

03Finalité et base légale

Les données transmises via formulaire sont utilisées uniquement pour répondre à votre demande (premier échange, proposition de mission). Elles ne sont jamais revendues, ne servent à aucune prospection, ne sont transmises à aucun tiers.

Base légale : votre consentement explicite, recueilli via case à cocher avant l'envoi.

04Durée de conservation

Les données issues des formulaires sont conservées 30 jours maximum si aucune mission ne fait suite, puis supprimées. Si une mission est engagée, elles basculent sous le régime du secret professionnel (voir section 06).

05Sécurité

Les échanges par formulaire sont transmis par e-mail chiffré en transit (TLS). Les données ne sont stockées sur aucun serveur tiers non conforme au RGPD. Aucune solution d'intelligence artificielle tierce n'a accès aux contenus de vos demandes.

06Secret professionnel et séances

Le contenu des séances relève du secret professionnel absolu. Aucune note identifiante n'est conservée au-delà du strict nécessaire opérationnel. Les notes de mission sont détruites à la clôture — ou remises à la personne accompagnée si elle le souhaite.

En mission entreprise, la confidentialité envers le commanditaire est contractualisée par convention tripartite. Le contenu des séances n'est jamais remonté ; seuls les indicateurs convenus sont partagés lors du point d'étape.

07Vos droits

Conformément au RGPD, vous disposez d'un droit d'accès, de rectification, d'effacement, de portabilité, de limitation et d'opposition sur vos données personnelles. Pour exercer ces droits : geoffbertrand@gmail.com.

Vous pouvez également introduire une réclamation auprès de la CNIL (cnil.fr).

08Outils tiers utilisés

Le site intègre les services tiers suivants :

  • Cal.com / Calendly (prise de rendez-vous) — conforme RGPD, données minimisées. Consulter leur politique avant de réserver.
  • Polices d'écriture auto-hébergées — aucune donnée transmise à des tiers.

Aucun autre service tiers n'a accès à vos données.

Voir également : mentions légales.
